# utca

Tools for analyzing city street networks in the framework of mosaics. Code for my thesis work
> The Analysis of the Evolution of the City Street Network

## Repository contents

* `utca.py`: main utilities module for analyzing street networks
* `save_osm_graphs.py`: script to save locally the street graphs of Hungarian towns
* `save_neat_graphs.py`: script to simplify the street graphs with neatnet
* `00_city_demo.ipynb`: notebook demonstrating the basic usage of the `utca` module
